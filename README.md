# NASA Spaceapps Challenge 2023

Frontend for a fire data representation web-app
Gets data from model hosted at [repository](#) and displays as different graph views

### Dependencies
1. Flask
  ```bash
  $ pip install FLask
  ```
2. TailwindCSS
3. FlowbiteJS
4. Charts.js
5. sqlite3

To run this application:
```python3
flask --debug run
```
